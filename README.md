# Logging

Some simple classes to aid in logging messages.

## Usage

Add the maven dependency:

    <dependency>
        <groupId>com.github.codemonstur</groupId>
        <artifactId>logging</artifactId>
        <version>1.0.0</version>
    </dependency>
